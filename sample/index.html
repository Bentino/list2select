How to use

  1. Include list2select.css and list2select.js in dist directory to head section of the HTML page

    Example

    <link rel="stylesheet" type="text/css" href="list2select/dist/list2select.css" charset="utf-8">
    <script src="list2select/dist/list2select.min.js" charset="utf-8"></script>

    *For the best performance the JS load script should place at the lowest position in the body section

  2. Insert breakpoint class into the container of the list element for example:

    <div class="list-container l2s-display-md">
      <ul> <-- The default selector is set to the list
        <li>list item 1>/li>
        <li>list item 2>/li>
        <li>list item 3>/li>
      </ul>
    </div>
    <-- You should specific selector to place the generated select list outside the container

Available options

Option name Description
nullOption Create null option for the select list
Default: FALSE
nullOptionLabel Change label for the null option
Default: "— Select an option —"
tabMode Convert tab menu to select
Default: FALSE
tabSelector Selector to the tab menu
Default: ""
toggleDisplayWidth Breakpoint to responsive
Default: "md"
insertBefore Method to place the generated select list
Default: FALSE
placeAt Custom selector to place the generated select list
Default: ""
childSelector Custom selector to the specific elements
Default: "li"
customClass Define custom classes to the generated select list
Default: ""
