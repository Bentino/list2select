List2Select demo

Please read.

This plugin (jQuery) use to transform <ul>,<ol> element to <select>, useful for responsible display with 2 mode (jump or tab mode). Compatible with Bootstrap™, Tab UI and Foundation™ Tab UI.

FAQ.

Why I should use this plugin?

When you want to work with responsive menu, basically you have 2 choice (or more)

  • First, deal with frontend framework (Bootstrap or Foundation) to construct element exactly as in document, nice look but spend more time and complex.
  • Second, switch to use <select> element, nice support with native mobile UI element (saved time and consistence mobile's user experience)
